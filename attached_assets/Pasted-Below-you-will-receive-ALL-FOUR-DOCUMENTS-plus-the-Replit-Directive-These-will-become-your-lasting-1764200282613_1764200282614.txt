Date: 26 November 2025
Prepared for: Replit Team / Development Environment
Action: Mandatory implementation verification for audit closure.

1. Mandatory Actions

Replit must confirm that:
	1.	All legal documents match final approved versions
	2.	Footer is updated across all portals
	3.	Registered office address is correct
	4.	Support email is correct
	5.	Mapping functionality is fully disabled everywhere
	6.	Lat/lng storage is fully null
	7.	CSP contains no mapping vendors
	8.	18+ age restriction is enforced
	9.	Email templates contain the correct footer
	10.	GDPR export and deletion operate as described
	11.	All changes are present in production build once published

2. Evidence Required

For each item above, Replit must provide:
	•	Full code snippet or diff
	•	File path
	•	Screenshot (UI, database, logs, headers, network)
	•	Terminal output where applicable
	•	Confirmation the feature exists in the production environment
	•	Confirmation of build ID or deployment timestamp

3. Final Deployment

Once changes are verified, Replit must:
	1.	Run production build
	2.	Publish the build
	3.	Return:
	•	Production URL
	•	CSP headers
	•	Screenshot of privacy/terms/cookie pages
	•	Confirmed mapping disablement in production

4. Acceptance

Audit will only close once all required evidence is returned.
